Dominion card game project

This project is common to all first year cs students at IUT Montpellier.

The project's goal is to code the dominion card game in java. For this project I was in a group with two other people. Each of us had the same level of implication we added, modified and corrected each others code. The programm should allow for multiple users to play and it should integrate with the graphical interface given by the teachers (CLI and WEB interface). For more information : https://github.com/IUTInfoMontp-M2103/Projet-Dominion The project was due for the 05.15.2019. I do not have the results yet but I'll update this section when I get them.
