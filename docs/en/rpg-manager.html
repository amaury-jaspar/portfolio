RPG Manager

What is it ?

RPG Manager is a tool to help you manage your table RPG games. There are Game Master (GM) and player specific interfaces.

This software is currently being built to be used with the Naheulbeuk RPG (A french rpg project), but it will also work with other types of games such as D&D. Because the software is open-source, you can add any features you might want for your specific game.

RPG Manager is written in JAVA and uses the GTK framework for the interface.

Functionalities

Exporting your games and characters

RPG Manager allows you to export your games and characters to XML files which can then be imported into RPG Manager to transfer games information or transfer characters between games.

The XML files contain all the important information related to your game or character such as name, inventory, abilities, players...

Status

The software is in a VERY EARLY stage. Most of the structure is figured out and the export to XML is working but there is still a lot of work to do, especially when it comes to the UI.
