Durbaton - A simple RPG Manager

What is it ?

Durbaton is a tool that allows you to manage your table RPG games. There are Dungeaon Master (DM) and player specific interfaces.

This software is currently being built to be used with the Naheulbeuk RPG (A french rpg project), but it will also work with other types of games such as D&D. Because the software is open-source, you can add any features you might want for your specific game.

Durbaton is written in Python and uses the GTK framework with PyGObject for the interface.

Status

The software is in a VERY EARLY stage. Most of the structure and UI layout is figured out but there is still a lot of work to do.

Features

Exporting your games and characters

Durbaton allows you to export your games and characters to XML files which can then be imported into Durbaton to transfer games information or transfer characters between games.

The XML files contain all the important information related to your game or character such as name, inventory, abilities, players...

Creating, editing and managing your games

When starting Durbaton you will be presented with a list of all your saved games, giving you the necessery information (name of the game and the dungeon master, number of players, current turn number). Selecting a game will give you more option and information. For example you will be able to edit the game's parameters, launch the game or delete it.
