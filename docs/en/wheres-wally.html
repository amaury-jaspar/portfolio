Where's Wally - Machine learning model with tensor flow

This is one of my group assignements for the third semester of my CS classes at IUT Montpellier.

The goal of the project is to train a machine learning model by feeding it Where's Wally (Waldo in the US) images and telling it, whether it has found Wally or not. We are going to use the tensor flow library for python. This will allow us to take advantage of the many ressources available for this library. We will also be able to use pre existing models in order to focus on the training aspect.

As this is our first dabble in the machine learning world, our primary focus is on trying to understand the training process. For now we are thinking about making different versions, each using a different approach (using already trained models, having an sort of image manipulation layer that would distort the image in different ways...). In the end we hope that we'll be able to achieve a success rate of about 60%.
